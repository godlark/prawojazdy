Proces generowania

Generowanie kart to złożony proces składający się z kilku etapów:

  1. Wczytywanie danych python df = pd.read_excel(exam_questions_xlsx)
  2. Odczyt pliku Excel z pytaniami
  3. Walidacja struktury danych
  4. Przygotowanie DataFrame

  5. Weryfikacja wymaganych kolumn

  6. Analiza pytań python if "Tak" in correct_answer or "Nie" in correct_answer: model = Model.MODEL_1 else: model = Model.MODEL_2

  7. Automatyczne wykrywanie typu pytania
  8. Przypisanie odpowiedniego modelu
  9. Walidacja odpowiedzi

  10. Przygotowanie dodatkowych pól

  11. Przetwarzanie mediów python media_type = get_media_type(media_path) if media_type == MediaType.Video: convert_to_webm(source_file, dest_path, crf_value)

  12. Wykrywanie typu mediów
  13. Konwersja do odpowiednich formatów
  14. Optymalizacja rozmiaru
  15. Prefixowanie nazw plików

Generowanie pakietu

  1. Tworzenie notatek python note = genanki.Note( model=model, fields=[ note_data.id, note_data.question, note_data.correct_answer, os.path.basename(note_data.media), *note_data.additional_fields ] )

  2. Dodawanie do talii python deck = genanki.Deck( random.randrange(1 << 30, 1 << 31), "Prawo Jazdy - teoria B" ) deck.add_note(note)

  3. Pakowanie python package = genanki.Package(deck, media_files=media_files) package.write_to_file("prawo_jazdy.apkg")
