Architektura systemu

PrawoJazdy to kompleksowy system generowania materiałów edukacyjnych, składający się z kilku kluczowych komponentów:

  1. Rdzeń systemu
  2. Generator talii Anki
  3. System zarządzania mediami
  4. Moduł konwersji formatów

  5. Szablony kart HTML/JS

  6. Zarządzanie zasobami

  7. Automatyczne pobieranie materiałów
  8. Weryfikacja integralności
  9. Optymalizacja mediów

  10. Zarządzanie pamięcią podręczną

  11. Interfejs użytkownika

  12. Interaktywne karty Anki
  13. Responsywny design
  14. System timerów
  15. Obsługa multimediów

Technologie

Projekt wykorzystuje nowoczesny stos technologiczny:

  1. Backend
  2. Python 3.8+
  3. pandas do przetwarzania danych
  4. genanki do generowania talii

  5. ffmpeg do konwersji mediów

  6. Frontend

  7. HTML5 dla szablonów kart
  8. JavaScript dla interakcji
  9. CSS3 dla stylizacji

  10. Wsparcie dla WebM

  11. Narzędzia

  12. Hatch do zarządzania projektem
  13. Git do kontroli wersji
  14. mypy do typowania statycznego
  15. pytest do testów

Struktura projektu

.
├── src/prawojazdy/           # Główny pakiet Python
│   ├── __about__.py        # Informacje o wersji
│   ├── conversion.py       # Konwersja mediów
│   ├── decks.py           # Generowanie talii
│   ├── main.py            # Główny moduł
│   ├── media.py           # Obsługa mediów
│   ├── notes.py           # Tworzenie notatek
│   ├── templates.py       # Generowanie szablonów
│   └── templates/         # Szablony HTML/JS/CSS
├── media/                  # Zasoby medialne
├── scripts/                # Skrypty narzędziowe
├── tests/                 # Testy
└── visuals/               # Zasoby wizualne
